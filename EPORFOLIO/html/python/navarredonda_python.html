NAVARREDONDA_PYTHON_RA125

1. A mini-summary of what to do in the challenge:

This challenge involved creating a Python script to calculate the entrance fee for the "Navarredonda" venue. The program asks for user information such as age, height, residence, and group type. It automatically determines the day of the week using the datetime module and simulates a membership status using the random module. Based on these factors, a complex logic of discounts and surcharges is applied to the base price of 45.00€, and the final result is displayed using custom ASCII art digits.

2. Two complex concepts that you have learned:

  • ASCII Art with Dictionaries: I learned how to use a Python dictionary (ASCII_DIGITS) to map characters like numbers and symbols to specific list patterns, allowing the program to print the final price in a stylized, large graphic format directly in the console.
  • Complex Conditional Logic: The script required implementing a detailed decision tree using if-elif-else structures to prioritize different pricing rules (e.g., granting free access to members over 65, or applying specific discounts for residents of Madrid vs. other provinces).

3. How have you felt:

I felt stimulated by the logic challenge. Translating real-world pricing rules into code requires careful attention to the order of conditions to ensure the correct price is calculated.

4. What would you improve about your solutions:

I would improve the input handling. Currently, if the user types "madrid" in lowercase or enters an invalid day type, the logic might fail. Adding input validation and normalization (e.g., .lower()) would make the script more robust.

5. Technical words in English for the glossary:

  • Datetime: A Python module that supplies classes for manipulating dates and times, used here to find the current day name (e.g., 'Friday').
  • Random: A module that implements pseudo-random number generators, used to decide if the user is a "Socio" or not.
  • f-string: Formatted string literals (prefixed with 'f') that allow embedding expressions inside string literals, used for formatting the price to two decimal places.
  • Float: A data type representing real numbers with a decimal point, used for the 'altura' (height) input.

6. What grade would you give the practice statement and why (0 to 10), and what would you improve:

I would give this practice a 9 out of 10. It is a great exercise for practicing control flow and dictionaries, though adding a requirement for loops (other than printing) could make it even more complete.
