TURTLE_PYTHON_RA123

1. A mini-summary of what to do in the challenge:

This practice focused on using the Python turtle library to create graphic representations. The challenge was divided into three parts: drawing the "La Salle" star using precise angles, writing my full name ("ALBERTO R.") by controlling the pen coordinates to draw each letter, and creating a "Creative" free-form drawing which resulted in a multi-colored spiral generated by a loop.

2. Two complex concepts that you have learned:

  • Pen Control & Coordinates (penup/setpos): In the name script, I learned how to move the cursor without drawing lines by using penup(), moving to a specific X,Y coordinate with setpos(), and then using pendown() to start drawing again. This was essential to create the spaces between the letters A, L, B, E, R, T, O.
  • Algorithmic Art & Randomness: In the creative script, I explored how to generate complex patterns using a loop. By iterating 207 times and changing the RGB color randomly with color(random.random(), ...) at each step, I created a visual spiral effect where the circle radius decreases dynamically.

3. How have you felt:

I felt very creative. Unlike standard coding tasks that output text, this allowed me to visualize the code logic immediately. Drawing the star required patience to get the angles right, but the result was rewarding.

4. What would you improve about your solutions:

I would improve the modularity of the "Nombre.py" script. Currently, the code for each letter is written sequentially. I should define functions like def draw_A(): or def draw_L(): so I can reuse the code to write different words easily, rather than hardcoding the movements for just one name.

5. Technical words in English for the glossary:

  • Turtle Graphics: A popular way for introducing programming to kids, providing a visual representation of the code execution.
  • Canvas: The window or area where the turtle draws the shapes.
  • Setpos (Set Position): A function that moves the turtle to an absolute coordinate (x, y) on the canvas.
  • Fill: The process of coloring the inside of a closed shape, initiated by begin_fill() and completed with end_fill().
  • Loop: A programming structure that repeats a sequence of instructions until a specific condition is met (used in the creative spiral).

6. What grade would you give the practice statement and why (0 to 10), and what would you improve:

I would give this practice an 8 out of 10. It is a fun introduction to Python, but I would add a requirement to use functions to encapsulate the logic for drawing repetitive shapes.
