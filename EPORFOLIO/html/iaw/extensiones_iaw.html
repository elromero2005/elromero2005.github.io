EXTENSIONES_VSC_IAW_RA1

1. A mini-summary of what to do in the challenge:

The challenge consisted of setting up a productive development environment in Visual Studio Code. The main task was to research, select, and install a specific set of extensions to enhance workflow for technologies like PHP, HTML, Markdown, and MySQL. The output involved documenting these choices in a README.md file, explaining the utility of each extension (such as Live Server for previews or PHP Debug for troubleshooting) and providing evidence of their installation.

2. Two complex concepts that you have learned:

  • Debugging with Xdebug: Through the PHP Debug extension, I learned about the concept of connecting the IDE to a PHP runtime to inspect code execution, rather than just relying on simple print statements.
  • IntelliSense/Language Servers: By using PHP Intelephense and MySQL Autocomplete, I understood how "code intelligence" works, providing context-aware completions, function definitions, and parameter hints to reduce syntax errors and speed up coding.

3. How have you felt:

I felt very productive and satisfied.

4. What would you improve about your solutions:

I would improve the documentation by adding specific configuration examples for the extensions. For instance, explaining how to configure the launch.json file for the PHP Debug extension or listing the most useful keyboard shortcuts for Markdown All in One would make the guide more complete and useful for future reference.

5. Technical words in English for the glossary:

  • Debugging: The process of identifying and removing errors from computer hardware or software.
  • IntelliSense: A code-completion aid that includes a number of features: List Members, Parameter Info, Quick Info, and Complete Word.
  • Snippet: A small region of re-usable source code, machine code, or text.
  • Live Reload: A feature that automatically refreshes the web browser when changes are saved in the file, as seen with Live Server.
  • Markup: A system for annotating a document in a way that is syntactically distinguishable from the text, such as Markdown or HTML tags.

6. What grade would you give the practice statement and why (0 to 10), and what would you improve:

I would give the assignment a 9 out of 10.
