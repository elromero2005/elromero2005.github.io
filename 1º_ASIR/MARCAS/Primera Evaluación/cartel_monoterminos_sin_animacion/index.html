Monotermino

1. ¿Qué etiqueta se utiliza para crear un enlace?
Respuesta: a
2. ¿Qué palabra clave se usa para declarar una variable?
Respuesta: var
3. ¿Cómo se llama la etiqueta utilizada para las listas desordenadas?
Respuesta: ul
4. ¿Qué tipo de relación se define cuando un campo de una tabla está relacionado con el campo principal de otra tabla?
Respuesta: ForeignKey
5. ¿Qué tipo de clave se utiliza para identificar de manera única cada registro en una tabla?
Respuesta: PrimaryKey
